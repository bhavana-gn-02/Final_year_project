    Auto Rickshaw Booking System, Dharwad..!


    AUTO RICKSHAW BOOKING SYSTEM

    It is an online application which is used to manage all auto work online. It is useful for auto rickshaw union/agency which are involved in providing the services for the customers. It is very easy for the customers to book the auto rickshaw online and rickshaw drivers are able to keep the track of the location of the customer. There is no online service at present in Hubballi Dharwad for auto rickshaws. The customer can use this application by entering his details.
    The main objectives of this project is:
    • The customer can view and book the auto rickshaw in online.
    • It reduces the time which is required while hiring the auto manually.
    • It allows the customer to reserve the auto rickshaw hence it manages the auto rickshaw drivers income.
    • It is cost effective as it minimizes the payment price.
    • It will simplify the task and reduce the manual work.
    • It keeps the various details of the user such as user location,booking details,driver details etc..
